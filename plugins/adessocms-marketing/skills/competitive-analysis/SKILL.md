# Wettbewerbsanalyse

Dieses Skill bietet Frameworks für systematische Wettbewerbsanalyse.

## Wettbewerber-Typen

### Direkte Wettbewerber
- Gleiche Produkte/Services
- Gleiche Zielgruppe
- Gleiche Preisklasse

### Indirekte Wettbewerber
- Andere Lösung, gleiches Problem
- Substitute
- Angrenzende Märkte

### Potenzielle Wettbewerber
- Start-ups im Bereich
- Große Player die einsteigen könnten
- Internationale Expansion

## Analyse-Framework

Für jeden Wettbewerber analysieren:

1. **Unternehmen**: Größe, Geschichte, Ressourcen
2. **Angebot**: Produkte, Preise, Features
3. **Positionierung**: Wie positionieren sie sich?
4. **Stärken**: Was machen sie gut?
5. **Schwächen**: Wo sind sie verwundbar?
6. **Strategie**: Wohin entwickeln sie sich?

## Porter's Five Forces

1. **Wettbewerbsintensität**: Wie stark ist der Kampf?
2. **Neue Anbieter**: Wie leicht ist Markteintritt?
3. **Substitute**: Welche Alternativen gibt es?
4. **Macht der Kunden**: Wie viel Macht haben Käufer?
5. **Macht der Lieferanten**: Wie abhängig ist man?

## Differenzierungsstrategien

- **Produktführerschaft**: Bestes Produkt
- **Kundenintimität**: Beste Beziehung
- **Operative Exzellenz**: Bester Preis/Effizienz
